2. Círculo Mínimo Envolvente
Problema

El Smallest Enclosing Circle (SEC) es el círculo de radio mínimo que contiene un conjunto de puntos. A diferencia del convex hull (polígono), el SEC es un círculo y tiene aplicaciones en:

    Colocación de antenas de telecomunicaciones (área de cobertura).
     Diseño de zonas de seguridad.
    Optimización de rutas de drones.

2.1 Implementación con algoritmo de Welzl

El algoritmo de Welzl es elegante y usa recursión + aleatorización:

función welzl(P, R):
    # P = puntos restantes, R = puntos en el borde (|R| ≤ 3)
    si P está vacío o |R| = 3:
        return círculo_trivial(R)

    p = punto aleatorio de P
    C = welzl(P \ {p}, R)

    si p está en C:
        return C
    else:
        return welzl(P \ {p}, R ∪ {p})

Su tarea:

    Implementa welzl(points) con:
        Función auxiliar trivial_circle(R) para casos base.
        Test de punto-en-círculo robusto.
        Manejo de casos degenerados (2 puntos colineales, etc.).

    Implementa enfoque determinista alternativo (para comparar):
        Usa el convex hull como optimización inicial.
        Prueba todos los pares y triples de puntos del hull.
        Selecciona el círculo mínimo.

    Experimentos:
        Compara tiempo de ejecución: Welzl vs. enfoque determinista
        Datasets: 10, 100, 1000, 10000 puntos
        ¿Cuándo Welzl es más rápido? ¿Por cuánto?

Escenario: Una empresa de telecomunicaciones quiere cubrir N barrios de Quito con una sola antena que minimice el radio de cobertura.

Dataset: Genera o usa coordenadas de 50-100 barrios de Quito (puedes usar datos reales de coordenadas de parroquias).

Tareas: 

    Calcula el círculo mínimo envolvente
    Visualiza: puntos + círculo + convex hull (superpuestos)
